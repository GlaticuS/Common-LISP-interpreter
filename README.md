Это неполный интерпретатор CLISP
Автор: Анна Гладких
На данный момент поддерживаются вычислительные операции +,-,/,* и функции car и cdr (работают просто для списка аргументов).
Поддерживается вывод атомов и строк (вводимых не в качестве списка)
(Прим.: сканирование имеет больше возможностей)

Запуск: python interpreter_all.py example.txt

Примеры для example.txt:
(+ 2 3 4 5) -> 14
(car 1 2 30) -> 1
(cdr 1 2 30) -> (2 30 )
1234 "hello world" -> 1234 \n hello world
(+ 2 3 5 (+ 2 3)) 123 -> 15

Баг:
Не совсем корректно работает с вложенными списками (стоит добавить какой нибудь символ (число, строка) после вложенных списков)
